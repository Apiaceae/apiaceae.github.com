Debian系统中，对VIM插件加载方式采用独特的管理方式，Vim-addon-manager。

此工具安装后默认会检测 /usr/share/vim/registry/ 下haml配置文件，如若存在插件同名配置文件或vim-rumtime配置文件，则加载并读取，随后可以使用 vim-addons list 显示，使用 vim-addons install 来安装。

假设需要安装 NERDTree 插件，步骤如下：

1. 下载插件文件，复制到 /usr/share/vim/addons/ 下

2. 在 /usr/share/vim/registry/ 中新建 NERDTree.haml 文件，内容如下：

# vim-addons list 显示的插件名称
addon: NERDTree
# 插件描述
description: “NERDTree is a file explorer”
# 文件位置，默认当前目录为Vim-addon-manager的源地址（可使用 vim-addons -s 设置）:
files:
- plugin/NERD_tree.vim
- doc/NERD_tree.txt

3. 在终端运行 vim-addons install NERDTree 完成安装。
